# 講師版教案

## 目的

這份教案不是給學員直接閱讀，而是幫講師在帶 06 時控制節奏、示範順序與提問方式，避免課程變成單純示範框架語法。

## 建議授課節奏

1. 先回顧 04 的 JWT API 與 05 的 Axios / JSP 脈絡。
2. 再說明為什麼現在需要 Vue SPA，而不是再做一個 HTML 測試頁。
3. 依序帶 Vue 基礎、元件責任、Router / Pinia、Quasar、資料抓取與錯誤狀態。
4. 再講 CORS、build / deploy、Vue / JSP 並存與 migration strategy。
5. 最後用 exercises 收尾，要求學員從架構與交付角度回答問題。

## 搭配文件

- demo-script.md：講師現場示範順序
- common-mistakes.md：學員常見錯誤與糾正方式
- module-walkthrough.md：逐模組教學重點與提問
