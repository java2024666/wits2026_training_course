# Module Walkthrough

## Module 01

- 目標：讓學員建立 Vue 3 + Vite + TypeScript + SFC 的基本心智模型。
- 提問：如果不用 SFC，元件化與型別化會卡在哪裡？

## Module 02

- 目標：讓學員理解 Options API 與 Composition API 的差異與元件拆分原則。
- 提問：什麼邏輯應該留在元件裡，什麼時候該抽成 composable？

## Module 03

- 目標：完成 JWT 登入、Pinia store 與 route guard 閉環。
- 提問：為什麼路由保護與後端 JWT 驗證是兩層不同的保護？

## Module 04

- 目標：把 Quasar、CORS 與 Vue / JSP 並存策略放進專題脈絡。
- 提問：為什麼企業常選擇漸進式整合，而不是全面前端重寫？

## Module 05

- 目標：建立 data fetching、form state 與 error handling 的分工。
- 提問：loading、error、empty state 為什麼不能只靠畫面補丁處理？

## Module 06

- 目標：建立 build、deploy、env 與 migration strategy 觀念。
- 提問：本機可跑卻部署失敗時，你第一個檢查的是哪一層？
