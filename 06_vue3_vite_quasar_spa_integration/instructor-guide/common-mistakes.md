# 學員常見錯誤

## 錯誤 1：把 Vue 課理解成純 UI 課

常見說法：我只要把畫面做出來就好。

講師修正方式：提醒學員本課重點是 JWT、Router、Pinia、CORS 與既有系統整合，不只是畫面。

## 錯誤 2：把 localStorage 當成完整狀態管理

常見說法：token 存 localStorage 就夠了。

講師修正方式：要求學員說明 Pinia 的角色，以及 route guard 為什麼不能分散在每個頁面自己判斷。

## 錯誤 3：把 CORS 當成前端 bug

常見說法：Axios 壞掉了。

講師修正方式：回到瀏覽器同源政策與後端設定責任邊界。

## 錯誤 4：認為現代化就等於全面重寫 JSP

常見說法：既然用 Vue，舊 JSP 全部砍掉就好。

講師修正方式：要求學員從成本、風險、交付節奏與系統穩定性回答。
