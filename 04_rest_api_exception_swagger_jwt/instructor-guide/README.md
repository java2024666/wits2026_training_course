# 講師版教案

## 目的

這份教案幫講師把 04 從「會做 Controller 與 JWT」提升成「能交付 API、能被前端與測試整合、能被排查」。

## 建議授課節奏

1. 先講 API contract 與回應格式。
2. 再講 exception handling。
3. 接著導入 Postman 與 OpenAPI。
4. 然後講 JWT 與 security 回應。
5. 最後收在 CORS、delivery readiness 與 exercises。

- demo-script.md：現場示範順序
- common-mistakes.md：學員常犯錯誤
- module-walkthrough.md：逐模組教學提問
