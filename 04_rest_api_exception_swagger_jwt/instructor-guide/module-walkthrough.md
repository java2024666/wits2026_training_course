# Module Walkthrough

## Module 01

- 目標：建立 API 資源模型與回應結構。
- 提問：這條 API 回答的是哪個業務需求？

## Module 02

- 目標：建立一致錯誤格式與例外分流。
- 提問：這個錯誤應該是 validation、business 還是 system error？

## Module 03

- 目標：用 Postman 與 OpenAPI 建立交付共識。
- 提問：文件與實作若不一致，誰會先出事？

## Module 04

- 目標：建立 JWT 基本流程與受保護 API。
- 提問：拿到 token 之後，如何證明保護真的生效？

## Module 05-07

- 目標：補強 contract 演進、security exception、CORS 與 delivery readiness。
- 提問：這題是 contract 問題、安全問題，還是交付流程問題？
