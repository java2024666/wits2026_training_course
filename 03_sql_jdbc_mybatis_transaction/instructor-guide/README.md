# 講師版教案

## 目的

這份教案幫講師把 03 的內容從「會教語法」升級成「能教資料一致性與真實排查」。

## 建議授課節奏

1. 先講資料一致性在金融系統的重要性。
2. 再帶 SQL 與資料模型。
3. 接著拆 JDBC 與手動 transaction。
4. 再導入 MyBatis 與 MyBatis Plus 的選型。
5. 最後收在 propagation、timeout、readOnly 與 exercises。

## 搭配文件

- demo-script.md：現場示範順序
- common-mistakes.md：學員常見錯誤與糾正方式
- module-walkthrough.md：逐模組提問與重點
